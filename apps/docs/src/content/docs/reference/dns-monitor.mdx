---
title: DNS Monitor
---

A DNS Monitor is a component that allows you to monitor the status of DNS records. It can be used to monitor the status of a domain name, a subdomain, or any other DNS record.

It will fetch the following DNS record types:
- A
- AAAA
- CNAME
- MX
- NS
- TXT

## Configuration

### URI 
The domain or subdomain you want to monitor e.g `openstat.us`

### Regions 

The regions where we are triggering the DNS request.
