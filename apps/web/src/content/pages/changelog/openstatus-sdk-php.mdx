The openstatus PHP SDK is now available. Use it to interact with openstatus programmatically from your PHP applications.

Install it with Composer:

```bash
composer require openstatus/sdk-php
```

Find the package on [Packagist](https://packagist.org/packages/openstatus/sdk-php) and the source on our [GitHub repository](https://github.com/openstatusHQ/sdk-php). If you have any questions or feedback, feel free to reach out.
