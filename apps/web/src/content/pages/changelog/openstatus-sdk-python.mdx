The openstatus Python SDK is now available. Use it to interact with openstatus programmatically from your Python applications, scripts, and automations.

Install it from PyPI:

```bash
pip install openstatus
```

Find the package on [PyPI](https://pypi.org/project/openstatus/) and the source on our [GitHub repository](https://github.com/openstatusHQ/sdk-python). If you have any questions or feedback, feel free to reach out.
