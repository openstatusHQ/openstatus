We've released a collection of [Agent skills](https://github.com/openstatusHQ/skills) to help you manage your status page without leaving your favorite terminal or as part of automated workflows.

Install them with:

```sh
npx skills add openstatushq/skills
```

What you'll get:

- **Incident communication**: write clear, empathetic status updates for every phase of an incident
- **Maintenance announcements**: draft scheduled maintenance notices from planning through completion
- **Status reports**: generate periodic health summaries covering uptime, incidents, and maintenance
- **Status page context**: configure your services, SLAs, and communication tone once and reuse across all skills
