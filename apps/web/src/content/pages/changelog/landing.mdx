## Monitoring as Code

Apply changes to your monitors directly from the terminal.

```bash
$ openstatus monitors apply
```

Use version control to keep track of all your changes.

```yaml
"openstat.us":
  active: true
  frequency: 1m
  kind: http
  name: openstat.us
  regions:
    - ams
    - iad
    - syd
  request:
    method: GET
    url: https://openstat.us
```
