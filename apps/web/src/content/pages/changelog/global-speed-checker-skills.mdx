Enable AI agents like Claude Code to check website performance from multiple global locations directly in your terminal.

Install it via npx skills:

```bash
npx skills add openstatushq/skills
```

Or with the Claude Code plugin system:

```bash
/plugin marketplace add openstatushq/skills
/plugin install openstatus-skills
```

**Repository:** [@openstatushq/skills](https://github.com/openstatusHQ/skills)
