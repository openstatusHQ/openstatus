Each row above represents one call from the MCP `2025-06-18` handshake. Click a row to see the exact JSON-RPC request openstatus sent and the response the server returned. Request headers are never stored — re-run the check with your own `Authorization` header if the server requires auth.

Reports expire after **7 days**. To monitor an MCP server continuously, see the [monitoring guide](/docs/guides/how-to-monitor-mcp-server).
